This folder contains all the data and jupyter notebooks pertaining to wellbore data manipulation.

The folder named WellBore_Data contains all the data needed to run the notebooks.

The Jupyter notebooks in the folder are (Should be read in this order):

1) Finding all Unique Wells

This notebook will parse through all the data in the folder WellBore_data and find the unique wells to help avoid counting reaccuring wells more than once.

2) Finding the Most Common Wells.

This notebook will compile a list that will contain all the unique wells in all the data along with information as to which files a given well occured in.
